2/8

The Realization

Average users won't babysit

I had to constantly push back:

  • Manually said "log it" to ensure things got stored
  • Watched tool call logs to verify agent was working
  • Pressed into new info the agent surfaced
  • Used prompt techniques ("give me estimates") because it jumped to generalizations

Normal users won't do any of this. Valuable information would get lost.

3/8

The Product Question

How does an agent push back on itself?

The core challenge

How do we grow an agent that pushes back on the right things when each user has a unique problem space?

Catch-all instructions like "be skeptical" or "retrace your steps" don't lead to good behavior.

The agent needs to learn what to challenge for this specific user.

4/8

The Discovery

Agents can self-identify mistakes

Asked the agent to identify how I'd pushed back on it. It outlined its own mistakes:

  • Overcorrection
  • Assumption without verification
  • Overcomplication
  • Absolutism

Then it proactively asked: "Can I encode these as guardrails?"

Potential feature

"Grow quicker for me" — agent auto-improves based on its own error patterns.

5/8

What Shipped

Architectural discoveries

  • Experiments paradigm — agent self-initiates structured experiments
  • Command palette paradigm — user invokes specific agent modes
  • Precise daily_log block — no context loss despite 90k window
  • Week + food indexes — guide archival lookup for pattern analysis
  • Skill loading architecture — tested neal_test_coach, presales_closer swaps

Also: installed Letta as PWA. Recommends for all LaunchMate users.

6/8

Key Learnings

What the experiment proved

1. Growth baseline = ~2 weeks of daily use. That's the investment before payoff kicks in.

2. Tool call density > message count. Tool calls mean the agent is working, not just chatting.

3. Agent proactivity self-report:

  • Pattern flaggingHigh
  • Auto-adding coping toolsHigh
  • Comparative analysisHigh
  • Forward questionsHigh
  • Experiment suggestionsModerate
  • Contextual remindersModerate
