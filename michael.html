Starting Point

Took a detour on purpose

Back from almost two weeks off—fully recharged, energy 10/10.

Instead of jumping straight into LaunchMate's founder agent, I did something different: built my own chronic condition management agent from scratch.

Used it multiple times daily for two weeks. Wanted to understand how agents actually grow by living it myself.

The Data

What two weeks of real use looks like

1,052
messages
203
tool calls

More activity in two weeks than LaunchMate's most active founder generated over 16 weeks.

But here's the insight: message count isn't the metric. Tool call density is. That's the real signal—constantly logging, updating memory, iterating. That's growth.

2/6

The Realization

Average users won't do this

I had to constantly push back—asking for specific estimates instead of broad guesses, watching tool calls to verify behavior, pressing into new expert info the agent surfaced.

Normal users won't do any of that.

The agent gave recommendations that didn't track with long-term patterns I knew about myself. Catch-all instructions like "be skeptical" don't fix this.

Core Problem
The agent needs to learn to push back on itself.
3/6

The Discovery

Agents can self-identify mistakes

The agent could identify its own error patterns—with specific examples:

  • Overcorrection
  • Assumption without verification
  • Overcomplication
  • Absolutism

It proactively asked if it could encode these as guardrails.

"Grow quicker for me" — the agent auto-improves by identifying its own error patterns.
