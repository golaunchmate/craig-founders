The Realization

Average users won't babysit

I had to constantly push back on my agent—asking for estimates, watching tool calls, pressing into new info.

Normal users won't do that.

The agent jumped to broad conclusions, gave recommendations that didn't track with patterns I knew about myself.

Key insight
Catch-all instructions like "be skeptical" don't fix this. The agent needs to push back on itself.
3/6

The Discovery

Agents can self-improve

The agent could self-identify its mistakes:

  • Overcorrection
  • Assumption without verification
  • Overcomplication
  • Absolutism

It proactively asked to encode these as guardrails.

Potential feature: "Grow quicker for me" — agent auto-improves based on its own error patterns.
